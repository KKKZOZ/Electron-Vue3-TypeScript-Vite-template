# Electron-Vue3-TypeScript-Vite-template

## Introduction
这是一个简单的`Electron` + `Vue3` + `TypeScript` + `Vite` 模板
拥有以下特色：
1. 全部依赖均使用最新项
2. 极其简单的项目结构
3. 支持 Electron下的文件使用typescipt
4. 支持动态模块热重载
5. Out of the box

## Quick Start

```bash
npm install
npm run electron:serve
```

## Directory Structure
```
├── electron
|   └── background.ts         Electron entry file
```
